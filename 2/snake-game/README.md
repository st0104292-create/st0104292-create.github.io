# 🐍 貪吃蛇遊戲

一個現代化的貪吃蛇遊戲，支援桌面和手機玩遊戲。

## ✨ 特色

- **音效系統** 🔊
  - 吃食物：上升音調
  - 遊戲結束：下降音調
  - 升級：雙音提示
  
- **手機支持** 📱
  - 觸摸滑動控制
  - 方向按鈕
  - 響應式設計
  
- **遊戲功能** 🎮
  - 分級難度系統
  - 最高分記錄
  - 暫停/繼續
  - 實時計分

## 🎮 如何玩

### 桌面操作
- **方向鍵** 或 **WASD** 控制蛇的方向
- **空格鍵** 開始/暫停遊戲
- **滑鼠按鈕** 開始、暫停、重新開始

### 手機操作
- **觸摸滑動** 在遊戲畫面上控制蛇的方向
- **方向按鈕** 點擊控制
- **按鈕** 開始、暫停、重新開始

## 📊 遊戲規則

- 吃掉紅色蘋果獲得 10 分
- 每吃 5 個蘋果升一級，遊戲速度加快
- 最高 10 級難度
- 撞到自己會遊戲結束
- 蛇會從畫面邊緣穿過來
